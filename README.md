# Natural Language Search

A lightweight library for natural language search functionality.

## Installation

```bash
npm install natural-language-search
# or
pnpm add natural-language-search
# or
yarn add natural-language-search
```

## Usage

```typescript
import greet from 'natural-language-search';

greet();
// Output: Hello from natural-language-search package!
```

## Development

### Setup

```bash
pnpm install
```

### Dev Server

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

### Linting & Formatting

```bash
pnpm lint
pnpm format
```
